# Session Tracker

Простое приложение для отслеживания рабочих сессий и задач.

## Функционал

### 1. Трекер сессий
  - Добавление рабочих сессий с временем начала/окончания
  - Отслеживание выполненных задач за сессию
  - Комментарии к сессиям
  - Статистика по сессиям

### 2. Трекер задач
  - Добавление ссылок на задачи
  - Оценка сложности от 1 до 10
  - Комментарии к задачам
  - Удаление задач
  - Статистика по задачам (общее количество, средняя сложность, распределение)

## API Endpoints

### Сессии
- `GET /api/sessions` - Получить все сессии (требуется токен)
- `POST /api/sessions` - Создать сессию (требуется токен)

### Задачи
- `GET /api/tasks` - Получить все задачи (требуется токен)
- `POST /api/tasks` - Создать задачу (требуется токен)
- `DELETE /api/tasks/<id>` - Удалить задачу (требуется токен)
- `GET /api/tasks/stats` - Статистика по задачам (требуется токен)

### Аутентификация
- `POST /api/register` - Регистрация
- `POST /api/login` - Вход
- `GET /api/health` - Проверка работы API
